# DeDuBa Documentation

DeDuBa is a deduplicating backup system (Perl → C# port) using content-addressable storage.

- C# → C++ → POSIX interop via a native shim
- SHA-512 hashing and BZip2 compression
- Custom compact metadata serialization (Sdpack/Sdunpack)

Refer to the API section for detailed class and method documentation.
